# 内容质量宪章 V1

## 目标
- 代表性：Top5 是高价值主题，不是热度噪声。
- 可证据：每个主题必须可追溯到来源。
- 可阅读：首屏短、分层清晰。
- 不重复：跨层摘要不打架。

## 验收阈值
1. Top5 至少 4 条具备正文证据。
2. 每条 Top5 必须提供至少 1 个可点击来源。
3. 邮件首屏 AI 快读总长度 120-180 字。
4. 单条主题要点不超过 70 字。
5. L1 总览与 L2 Top5 语义重复率 < 20%。
6. RSS 重点速览每条 <= 60 字，包含来源信息。

## 失败模式（来自真实反馈）
- Top5 出现大量“暂无正文摘要”。
- AI 摘要没有落到正文事实。
- 首屏总结与下文重复冲突。
- 信息源不可见、不可验证。
- RSS 与热榜混杂，主次不清。
